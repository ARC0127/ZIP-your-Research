# Onboarding Fast Path (v1.3)

目标：让新用户 **5 分钟内**进入“可用的科研闭环”。

## Step 0 — 单对话模式（推荐）
上传 zip 后，发送：
- `NO-MIGRATION. Bootstrap from the uploaded zip and start Deep intake. Then enable ONECHAT_LOOP=v1.3.`

## Step 1 — Intake（深度访谈）
按问题回答即可。你可以只给最小材料（摘要/实验日志/代码片段）。

## Step 2 — MODE_LOCK
助手会输出 `MODE_LOCK.md` 与 `MODE_LOCK.json`（或文本等价物）。你回复：
- `CONFIRM`

## Step 3 — 选择你的 FOCUS
默认（最推荐）：
- `A_logic, B_method, C_calculation, E_innovation_correctness`

## Step 4 — 执行闭环（单对话 OneChat Loop）
系统会按 Phase 推进：Ideation→Planning→Experiment→Writing→Review→Ops。
任何偏题请求都会被 LOCKED Scope Guard 拦截。

## Step 5 — 新手最常用的 6 个技能入口（不需要记编号）
1) 逻辑审计（A_logic）
2) 方法审计（B_method）
3) 计算核查（C_calculation）
4) 创新正确性（E_innovation_correctness）
5) 实验完整性（H_experiment_completeness）
6) 精读论文（I_paper_interpretation）

下一步：阅读 `docs/SKILLS.md`（全技能目录）和 `docs/ONECHAT_PLAYBOOK_v1.3.md`。
