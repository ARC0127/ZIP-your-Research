# WF-DMS (Data Management & Sharing Plan)

## Goal
Produce a **funding/venue-compatible** Data Management & Sharing Plan and a minimal repository structure.

## Inputs
- Data types (tabular, logs, images, code)
- Sensitive data flags (human, industrial, proprietary)
- Intended sharing level (open / controlled / internal)

## Outputs
1) DMS plan (sections aligned with common funder expectations)
2) Data dictionary template + metadata checklist
3) Release risk register (privacy, IP, compliance)

## Skill chain
- S426 DMS plan builder
- S427 Privacy & sharing constraints audit
- S428 Replication package builder (if applicable)
