# WF-SYSREVIEW (Systematic Literature Review)

## Goal
A reproducible literature review pipeline: queries → screening → synthesis → related work narrative.

## Outputs
- PRISMA-style protocol draft
- Screening rubric + extraction table template
- Synthesis outline (themes, gaps, contradictions)

## Skill chain
- S238 Systematic review protocol
- S239 Screening rubric builder
- S234 Novelty search protocol (optional)
